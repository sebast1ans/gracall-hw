# gracall-hw

A blog management prototype app for a job application.

## Project setup

```
npm install
```

This prototype uses a simple local database, don't forget to run this:

```
npm run db:serve
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
